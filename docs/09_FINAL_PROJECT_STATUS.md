# Final Project Status

## Project Name

**RCDOK Certificates Site Project**

## Project Description

This project is an OCR-assisted web prototype for digitizing historical Philippine certificate records.

The system allows users to upload scanned certificate images. It then uses OCR to extract text, detects the certificate type, performs structured extraction, and generates a digitalized certificate document for review.

The project focuses on birth, death, and marriage certificates.
